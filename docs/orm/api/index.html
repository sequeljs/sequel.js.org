Sequel.js - ORM#

@sequeljs/orm is an ORM built on top of @sequeljs/ast and @sequeljs/omi for JavaScript.

Installation#

npm install --save @sequeljs/orm

Contributing#

Feel free to contribute with this library. For more information, take a look at our contributing guide.

License#

MIT

Inspiration#

@sequeljs/orm is basically a rewrite of Rails' Active Record library in TypeScript.
